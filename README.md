# rocket-yew-rust
## Description
Rebuilding Personal website in Rust

## Install and Run

Clone Repository
```git clone https://github.com/joshespi/rocket-yew-rust.git```

in the backend directory
```cargo watch -x run```

in the ui directory
```trunk watch```
